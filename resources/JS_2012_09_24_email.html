Optional exercises for this week

  • build a functionMAXthat that takes two arguments and returns the larger of the two
  • build a functionMINthat that takes two arguments and returns the smaller of the two
  • have your page ask for two numbers and write the max and min out to the page
  • EXTRA CREDIT - write theMINfunction using only a call to theMAXfunction
  • write a function that takes a number and returns true if it's even, false otherwise. Create a page that takes a number as input and writes out if the input is even or odd on the page.
  • EXTRA CREDIT -write a function that takes a character and returns true if that letter is a vowel, false otherwise. Create a page that takes input and returns if it's a vowel or consonant.
